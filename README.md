# Observability Stack

Bundled full observability suite for local dev

Contains:
- Grafana
- Prometheus
- Promtail
- Loki (Logs)
- Tempo (Tracing)
- Pyroscope (Profiling)

## Default Dashboard
Grafana serves on `localhost:3000`, there is a provided json to import that will give
self-observability (i.e. sense check that everything is working)
